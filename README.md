# Croppy x64

Croppy turns shipping-label PDFs into print-ready 4x6 labels for thermal printers.

Use the production app at [croppy.joost.io](https://croppy.joost.io/).

What it does:

- Upload one or more PDF labels.
- Auto-rotates pages
- Auto-detect or manually choose a crop preset.
- Export a cleaned-up 4x6 PDF that is easier to print.

Croppy can also be installed as a PWA from Chrome or Edge, which makes the workflow smoother: install it once, then open label PDFs directly in the app.

This project is AI-generated slop. It just happens to be useful for me and maybe you too.
